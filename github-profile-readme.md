Engineer, designer, and founder with 10 years across full-stack development, UX, growth, and user acquisition. Most of my hands-on work is TypeScript and React, with more recent work in Python, C++, and LLM tooling. I've founded and sold three products and spent the last decade running WithSeismic, delivering product, design, and engineering work for companies including Contra, Framer, The Motley Fool, and MIT.

Lately a lot of agentic workflows, ~~MCP servers~~ CLI tools, and game internals.

London, UK / Prague, CZ.
