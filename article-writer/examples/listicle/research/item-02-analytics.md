# Research: PostHog - Product Analytics

## Key Facts
- All-in-one: product analytics, web analytics, session replay, error tracking, feature flags, experimentation, surveys, data warehouse, CDP
- Open-source under MIT license, self-hostable via Docker/Kubernetes
- Usage-based pricing, no per-seat fees
- Free tier: 1M events + 5K replays/month
- 98% of customers use PostHog for free
- Most teams pay $150-$900/month
- 190,254+ teams, 13,443+ companies using it
- Notable users: Airbus, ElevenLabs
- Y Combinator backed

## Trade-offs
- Self-hosted version requires significant infrastructure for large scale
- Cloud version costs can grow with heavy usage
- Newer than Amplitude/Mixpanel in enterprise features
